Développeur d'applications mobiles

Mission :

  • Réalise une application à partir d"un cahier des charges pour mobiles, smartphones, tablettes, etc.
  • Choisit les logiciels et code spécifiques pour la construire
  • Peut également concevoir des versions mobiles de sites web existants
  • Peut mettre à jour des applications
  • Effectue des tests et corrige les éventuels problèmes techniques

Compétences :

  • Veille technologique
  • Maîtriser l'anglais technique
  • Connaissance de l'univers mobile et des normes (W3C, HTML5, CSS, etc.)
  • Connaissance des langages de programmation (Objective-C, Swift, Java, etc.)

Savoir-être :

  • Curieux
  • Autonome
  • Pédagogue
  • Créativité
  • Logique
  • Curiosité
  • Organisation

Formations :

  • Bac +2 : BTS Informatique
  • Bac +3 : BUT Informatique, licence professionnelle
  • Bac +4/5 : Université, Ecole d’ingénieurs (informatique)

Rémunération :

Le développeur d'applications mobiles peut gagner entre 2 900 € et 5 000 € brut par moi selon son expérience.
