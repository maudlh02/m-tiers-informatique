Développeur FullStack

Mission

  • Architecte des sites web
  • Intervient sur tous les aspects d'un projet informatique
  • Gère toutes les demandes informatiques de ses clients et est capable de concevoir un programme et de le maintenir
  • Travaille sur le back-office, pour régler les fonctionnalités de l'interface administrateur du site
  • Développe les fonctionnalités côté front-end avec la mise en page
  • Participe au codage et aux tests

Compétences :

  • Connaissances techniques très larges
  • Bonne compréhension business et marketing
  • Veille technologique
  • Compétences techniques à avoir :
    • Front : HTML, CSS, Javascript (et quelques un de ses frameworks (Angular, Vue.JS, React.JS, etc.))
    • Back : PHP (et quelques frameworks : Laravel, Symfony, etc.), Java, etc.
    • Bases de données : MySQL, PostgreSQL, MongoDB, etc.
    • Outil de gestion de projet : GitHub, GitLab, Trello, etc.
    • Design : connaissances en UX nécessaires pour permettre de rendre les sites web accessibles

Savoir-être

Le développeur FullStack doit posséder certaines aptitudes pour pouvoir exercer au mieux son métier :

  • Autonomie
  • Polyvalent
  • faire preuve de rigueur
  • Créatif
  • Rigoureux
  • Organisé

Formations qui permettent d'accéder à ce métier :

Plusieurs parcours sont possibles pour pouvoir exercer le métier de développeur FullStack :

  • Niveau Bac + 2 : BTS informatique
  • Niveau Bac + 3 :
    • Licence professionnelle
    • BUT informatique
  • Diplôme de type Bac +5 en informatique :
    • Master en informatique
    • Diplôme d'ingénieur en informatique

Comme beaucoup de nouveaux métiers, il y a aussi nombreux autodidactes passionnés par l’informatique qui exercent ce métier.

Rémunération :

Le salaire d’un développeur FullStack débutant se situe entre 30 000 € à 45 000 € par an, pouvant aller jusqu'à 70 000 € par an pour un profil sénior.
